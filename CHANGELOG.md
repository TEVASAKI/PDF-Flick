## [1.0.0] - 2026-03-02

### 初版リリース

#### 追加機能

- **PDFファイル一覧表示**: ダウンロードフォルダ内のPDFファイルを自動検出し、一覧表示
- **PDFプレビュー表示**: 各PDFの最初のページをプレビュー表示（200 DPI、PNG形式）
- **右フリック操作**: PDFカードを右にフリックしてファイルを保存フォルダに移動
- **左フリック操作**: PDFカードを左にフリックしてファイルを削除
- **Undo機能**: 「元に戻す」ボタンで直前の操作を取り消し
- **保存先フォルダ設定**: 初回起動時にファイル保存先フォルダを指定
- **操作履歴表示**: 処理済みファイル数（保存・削除）を表示
- **完了画面**: すべてのファイル処理後に整理完了画面を表示

#### デザイン

- **エレガント・プロフェッショナル型**: 日本的な余白の美学とモダンミニマリズムを融合
- **カラーパレット**: 白背景、墨色テキスト、深い緑（保存）、薄い紅色（削除）
- **タイポグラフィ**: Noto Serif JP（見出し）、Noto Sans JP（本体）
- **非対称レイアウト**: 黄金比を意識した配置で自然なフリック方向認識

#### 技術スタック

- React 19 + TypeScript
- Tailwind CSS 4
- shadcn/ui
- Vite
- pnpm

#### ドキュメント

- README.md: プロジェクト概要と使用方法
- SPECIFICATION.md: 詳細な機能仕様
- INSTALLATION_GUIDE.md: セットアップと導入手順
- CHANGELOG.md: 更新履歴（本ファイル）

## バージョニング方式

このプロジェクトは [Semantic Versioning](https://semver.org/lang/ja/) に従います。

バージョン番号は `MAJOR.MINOR.PATCH` の形式です。

- **MAJOR**: 互換性を破る変更
- **MINOR**: 新機能追加（互換性あり）
- **PATCH**: バグ修正

## 更新方法

### 開発環境での更新

最新版のコードを取得してください。

```bash
git pull origin main
pnpm install
pnpm dev
```

### 本番環境での更新

新しいバージョンをビルドしてデプロイしてください。

```bash
git pull origin main
pnpm install
pnpm build
# dist/ ディレクトリを本番サーバーにアップロード
```

## 既知の問題

### バージョン 1.0.0

- 大規模なPDFファイル（100MB以上）のプレビュー生成に時間がかかる場合があります
- 一部の特殊なPDF形式（スキャン画像のみ等）ではプレビューが正しく表示されない場合があります
